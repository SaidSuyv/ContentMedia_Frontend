## Development server

To start a local development server, execute the following steps:

1. Clone repository
```bash
git clone https://github.com/SaidSuyv/ContentMedia_Frontend
```
2. Install dependencies
```bash
cd ContentMedia_Frontend
npm install
```
3. Execute Angular Serve Command
```bash
ng serve
```
